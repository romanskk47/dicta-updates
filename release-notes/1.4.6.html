Dicta 1.4.6

What's New

  • Universal binary — now runs natively on both Apple Silicon (M1/M2/M3/M4) and Intel Macs.
  • Drag & drop from History — drag transcriptions from History browser or the latest preview card into any app.
  • File access recovery — clear error messages and a "Choose file…" button when macOS blocks file access (e.g., Voice Memos).
  • History browser polish — unified glass background, refined column divider, and consistent opacity tokens.
