Dicta 1.2.0

Added

  • Periodic license validation with a 24h retry window before deactivation
  • History browser window with split list/detail layout
  • Editable transcriptions with Cmd+S save and unsaved-changes prompts
  • Source metadata (URL/path) with Open/Copy actions in history detail
  • History actions: context menu, swipe-to-delete, Delete key

Changed

  • Activation now uses email sign-in codes (no link required)
  • License transfers to another Mac when you sign in there (one active device at a time)
  • History now opens in a dedicated window and closes the popover
