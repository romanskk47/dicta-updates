Dicta 1.3.3

Changed

  • Whisper long-form defaults — Background streaming now disables cross-chunk prompt conditioning by default to reduce premature endings on long dictations.

Fixed

  • Transcription tail clipping — Chunk merge now falls back to text merge when word timestamps are incomplete, preventing dropped endings.
  • Low-volume final tail loss — Final chunk is now processed even when RMS is below silence threshold; silence skipping remains for non-final chunks.
