Dicta 1.1.0

New Features

  • File transcription - import audio/video files (MP3, WAV, M4A, MP4, MKV, and more)
  • URL transcription - paste YouTube, Vimeo, or other video links to transcribe
  • FFmpeg + yt-dlp integration for extended format support

Improvements

  • Fixed YouTube download issues (403 errors) for certain videos
  • Skip silent audio chunks to prevent Whisper hallucinations
  • Improved permission handling and error messages
  • Better progress reporting for file/URL transcription

Technical

  • Binary auto-download (ffmpeg, yt-dlp) on first use
  • Max 4 hours per file
