Dicta 1.5.0

What's New

  • Email-first trial: new users now sign up with email during onboarding for a smoother 14-day trial experience.
  • Payment issue alerts: proactive in-app banner when a payment problem needs attention.
  • Migration support: existing users get a guided transition to the new sign-in flow.

Under the Hood

  • Architecture overhaul: recording state machine, dedicated permission manager, API client extraction, and view decomposition for a cleaner, more maintainable codebase.
  • Shared checkout logic across activation, trial-expired, and settings views.
